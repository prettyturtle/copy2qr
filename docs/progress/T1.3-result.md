```
HashRouter
  └── Routes

              ├── Route index (element=<HomePage />)   → /#/
              └── Route path="share" (element=<SharePage />)  → /#/share
```

- `import { HashRouter, Routes, Route } from 'react-router'` (react-router-dom 아님)

