### 검증 결과
- 디자인 셀프체크: PASS (기존 Button 컴포넌트 재사용, 기존 토큰 시스템 준수)
- 새 테스트: SKIP (테스트 미생성)
- 기존 테스트 회귀: SKIP (기존 테스트 없음)
- TypeScript: tsc --noEmit 통과 (오류 없음)

### 구현 상세
- Props: `onCopyLink`, `onDownloadQR`, `onReset` (각 `() => void`), `disabled?: boolean` (기본 false)
- 버튼 구성: "링크 복사" (primary, flex-1), "QR 다운로드" (secondary, flex-1), "초기화" (ghost, sm:flex-none)
- 반응형: `flex-col sm:flex-row` — 모바일 세로 스택, 태블릿+ 가로 배열
- disabled 범위: `onCopyLink`, `onDownloadQR` 버튼만 비활성화. `onReset`은 항상 활성 (초기화는 빈 상태에서도 동작해야 함)
- 접근성: `role="group"` + `aria-label="공유 액션"`, 각 버튼에 `aria-label` 부여
